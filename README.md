# MASI-FER: Memory-Attentive Swarm Intelligence for Facial Expression Recognition

This repository contains the implementation of the MASI-FER framework.

## 🚀 Features
- ResNet-50 + ASF module
- BiLSTM + Multi-Head Attention
- QIFABC Swarm Optimization
- Explainable AI (Grad-CAM, SHAP, LIME)
